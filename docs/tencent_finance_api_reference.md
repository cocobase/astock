# 腾讯财经股票数据接口参考文档

## 接口概述

| 项目 | 说明 |
|------|------|
| **接口地址** | `https://qt.gtimg.cn/q=` |
| **请求方式** | GET |
| **返回格式** | JavaScript 变量赋值格式 |
| **编码格式** | UTF-8 |
| **更新频率** | 实时（交易时间内约3秒刷新） |

## 返回数据格式

### 格式说明

返回数据为 JavaScript 变量赋值格式，每只股票对应一个变量：

```javascript
v_代码="字段1~字段2~字段3~...";
```

### 返回示例

**A股返回：**
```javascript
v_sh600519="1~贵州茅台~600519~1460.00~1459.88~1459.54~...";
```

**港股返回：**
```javascript
v_hk00700="100~腾讯控股~00700~489.200~496.600~...";
```

**美股返回：**
```javascript
v_usAAPL="200~苹果~AAPL.OQ~255.92~255.63~...";
```

**基金返回：**
```javascript
v_jj110022="110022~易方达消费行业股票~0.0000~0.0000~~3.1300~3.1300~-1.2930~2026-04-03~";
```

## 历史K线数据接口

### 接口说明

腾讯财经提供历史K线数据接口，支持A股、港股、美股的历史行情查询。

### A股/港股历史K线

**接口地址：**
```
https://web.ifzq.gtimg.cn/appstock/app/fqkline/get
```

**请求参数：**
| 参数位置 | 说明 | 示例 |
|----------|------|------|
| 第1个 | 股票代码 | `sh600519`, `hk00700` |
| 第2个 | 周期 | `day`, `week`, `month` |
| 第3个 | 开始日期 | `2024-01-01` 或留空 |
| 第4个 | 结束日期 | `2025-04-06` 或留空 |
| 第5个 | 周期数 | `100` |
| 第6个 | 复权类型 | `qfq`(前复权), `hfq`(后复权), 留空(不复权) |

**请求示例：**
```
# A股不复权日线
https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sh600519,day,2024-01-01,2025-04-06,100

# A股前复权日线
https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sh600519,day,2024-01-01,2025-04-06,100,qfq

# 港股前复权周线
https://web.ifzq.gtimg.cn/appstock/app/hkfqkline/get?param=hk00700,week,2024-01-01,2025-04-06,50,qfq
```

### 美股历史K线

**接口地址：**
```
https://web.ifzq.gtimg.cn/appstock/app/usfqkline/get
```

> ⚠️ **重要提示**：美股历史数据**仅支持前复权**（qfq），不支持不复权和后复权

**美股代码格式：**

| 交易所 | 后缀 | 示例 |
|--------|------|------|
| 纳斯达克 | `.OQ` | `usAAPL.OQ`, `usNVDA.OQ` |
| 纽约证券交易所 | `.N` | `usJPM.N`, `usBRK_B.N` |
| 美国证券交易所 | `.AM` | - |

**请求示例：**
```
# 美股日线（前复权）
https://web.ifzq.gtimg.cn/appstock/app/usfqkline/get?param=usAAPL.OQ,day,2025-01-01,2025-04-06,100,qfq

# 美股周线（前复权）
https://web.ifzq.gtimg.cn/appstock/app/usfqkline/get?param=usTSLA.OQ,week,2024-01-01,2025-04-06,50,qfq

# 美股月线（前复权）
https://web.ifzq.gtimg.cn/appstock/app/usfqkline/get?param=usMSFT.OQ,month,2024-01-01,2025-04-06,15,qfq
```

## 错误处理

### 无效代码返回

当请求的股票代码不存在时，返回：

```javascript
v_代码="v_pv_none_match="1";";
```

### 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `v_pv_none_match` | 股票代码不存在或格式错误 | 检查代码格式是否正确 |
| 空返回 | 网络问题或接口限制 | 稍后重试 |

## 使用示例

### Python 解析示例

```python
import requests

# 请求数据
url = "https://qt.gtimg.cn/q=sh600519,hk00700,usAAPL"
response = requests.get(url)
response.encoding = 'utf-8'

# 解析返回数据
data = response.text
# 按行分割
lines = data.strip().split(';')

for line in lines:
    if not line.strip():
        continue
    # 提取变量名和值
    if '=' in line:
        var_name, var_value = line.split('=', 1)
        code = var_name.replace('v_', '')
        # 去除引号
        value = var_value.strip().strip('"')
        # 按~分割字段
        fields = value.split('~')
        print(f"代码: {code}")
        print(f"名称: {fields[1]}")
        print(f"价格: {fields[3]}")
        print("---")
```

### JavaScript 解析示例

```javascript
// 假设返回数据已获取
const data = `v_sh600519="1~贵州茅台~600519~1460.00~..."`;

// 提取数据
const match = data.match(/v_(\w+)="(.+)"/);
if (match) {
    const code = match[1];
    const fields = match[2].split('~');
    
    const stock = {
        code: fields[2],
        name: fields[1],
        price: parseFloat(fields[3]),
        prevClose: parseFloat(fields[4]),
        open: parseFloat(fields[5]),
        change: parseFloat(fields[31]),
        changePercent: parseFloat(fields[32])
    };
    
    console.log(stock);
}
```
